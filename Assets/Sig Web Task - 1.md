### **Task 1: Enhance Your Webpage with CSS**  
#### **Objective:**  
Now that you've built a basic personal webpage using HTML, it's time to style it using **CSS**. Your task is to enhance your webpage from Task 0 by adding styles to make it visually appealing while maintaining a clean and structured layout.  

---

### **Requirements:**  
1. **Link an External CSS File:**  
   - Create a separate CSS file (`styles.css`) and link it to your HTML file using the `<link>` tag in the `<head>` section.  

2. **Apply Basic Styling:**  
   - Add background color and font styles to the entire page.  
   - Use different colors and font sizes for headings, paragraphs, and links.  

3. **Style the Navigation Bar:**  
   - Make the navigation bar horizontal.  
   - Add hover effects for links.  

4. **Enhance the Image Section:**  
   - Set a fixed width and height for the image.  
   - Use `border-radius` to make the image rounded if desired.  

5. **Improve Lists and Contact Section:**  
   - Customize the bullet points or numbering style of the list.  
   - Style the email and phone number links to look more interactive.  

6. **Footer Styling:**  
   - Ensure the footer is at the bottom of the page.  
   - Add a background color and center the text.  

---

### **Deliverables:**  
- A well-structured **HTML file** linked with an **external CSS file**.  
- Submit a **GitHub repository link** containing both HTML and CSS files.  
- Attach a **screenshot of the styled webpage** in the submission email.  

---

### **Learning Outcomes:**  
- Understanding how to separate structure (HTML) from presentation (CSS).  
- Learning basic **CSS properties** for text, background, layout, and styling.  
- Implementing a **navigation bar** with styling.  
- Enhancing webpage **readability and aesthetics**.  

---

This task builds on **Task 0** by introducing **styling concepts** without making it too complex. It also ensures that you start developing **good coding habits** by keeping **CSS separate from HTML**.
